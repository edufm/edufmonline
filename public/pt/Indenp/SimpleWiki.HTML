Simple Wiki

Participantes: Eduardo Magalhães, Leonardo Stayner

Local: Insper, 9° Semestre, Disciplina Natural Language proessing

Duração: 2 semanas

Formato: Desenvolvimento de um scrapper e simplificador de deinições da wikipedia.

Descrição: Este projeto visa utilizar a API da Wikipédia e a biblioteca NLTK para criar uma versão simplificada da Wikipédia, com poucas frases para cada definição.
