Robot Sim

Participantes: Eduardo Magalhães

Curso: Robotica e Automação Industrial

Período: 9° Semestre

Contato com o usuário: Não

Formato: Um Módulo em Python desenvolvido para simulação e animação de robôs

Duração: 4 semanas

Descrição: Um Módulo em Python desenvolvido para simulação e animação de robôs utilizando a matrix de Denavit-Hatemberg como base para a simulação e calculo dos movimentos e graus de liberdade dos robos criados pelos usuarios
