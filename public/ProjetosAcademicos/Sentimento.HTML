Analise de Sentimento

Participantes: Eduardo Magalhães, Bruno Viana, Hugo Mendez

Curso: Machine Learning

Período: 7° Semestre

Contato com o usuário: Não

Formato: Utilizar Tweets para fazer análises de sentimento e correlacionar notícias com os retornos do Ibovespa

Duração: 4 semanas

Descrição: Uma pesquisa cientifica e seu desenvolvimento pratico para verificar como diferntes algoritimos de NLP conseguem filtrar dados de tweets de influenciadores/fontes de noticia para gerar um dataset que, através de outra camada de algoritimos de machine learning, poderia correlacionar os dados aos movimentos do Ibovespa
