Analise de Sentimento

Participantes: Eduardo Magalhães, Bruno Vianna, Hugo Mendes

Local: Insper, 7° Semestre, Disciplina Machine Learning

Duração: 4 Semanas

Formato: Utilizar Tweets para fazer análises de sentimento e correlacionar notícias com os retornos do Ibovespa

Descrição: Uma pesquisa cientifica e seu desenvolvimento pratico para verificar como diferentes algoritimos de NLP conseguem filtrar dados de tweets de influenciadores/fontes de noticia para gerar um dataset que, através de outra camada de algoritimos de machine learning, poderia correlacionar os dados aos movimentos do Ibovespa
